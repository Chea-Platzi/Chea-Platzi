## 🚀 Getting started

If you want to run the site locally, you can:

1. Clone the project.
2. Go to project's folder

```bash
cd Tech-Deal
```

3. Install the dependencies

```bash
npm install
```

4. Run development enviroment.

```bash
npm run dev
```
